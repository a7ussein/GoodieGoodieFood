# Goodie Goodie Food 🍓👑

**Goodie Goodie Food** — Premium artisan desserts for your next event.

A mobile dessert truck serving Lewiston, Maine and surrounding areas. We bring fresh chocolate-dipped strawberries, fluffy mini pancakes, and crafted drinks directly to your celebrations.

## Features

- 🍫 **Menu Showcase** — Chocolate Strawberries, Mini Pancakes & Crafted Drinks
- 📅 **Online Booking** — Request a booking directly from the website
- 📱 **Fully Responsive** — Looks great on desktop, tablet, and mobile
- ✨ **Premium Design** — Smooth animations and modern aesthetics

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts (Fredoka, Quicksand)
- No frameworks or dependencies

## Getting Started

Simply open `index.html` in your browser — no build step required.
